# Income Tax Calculation Project

This project aims to provide an easy-to-use income tax calculation tool for individuals based on their income and applicable tax rates. It offers a simple interface, making it accessible for all users to estimate their tax obligations.

## Features
- Calculate income tax based on different slabs
- Configurable tax rates through a configuration file
- Easy to use

## Installation
To get started, clone the repository and install the required packages using the `requirements.txt` file.

## Usage
Run the `income_tax_calculator.py` script, and follow the prompts to input your income and get the calculated tax.
